Fonctions pour les régions :
Affichage et contenu :
showRegionInfo(event, region) - Fonction principale pour afficher les informations d'une région
getRegionImages(region) - Récupère la liste des images d'une région
getDefaultRegionImage(region) - Récupère l'image par défaut d'une région
getRegionTables(region) - Récupère la liste des tables aléatoires d'une région
getDefaultRegionTable(region) - Récupère la table par défaut d'une région
getRegionJsonTables(region) - Récupère la liste des tables JSON d'une région
getDefaultRegionJsonTable(region) - Récupère la table JSON par défaut d'une région
Mode édition :
enterRegionEditMode() - Active le mode édition pour une région
updateImageTabForRegionEdit(region) - Met à jour l'onglet images en mode édition
updateRegionTextTabForEdit(region) - Met à jour l'onglet texte en mode édition
updateRumeursTabForRegionEdit(region) - Met à jour l'onglet rumeurs en mode édition
updateTraditionTabForRegionEdit(region) - Met à jour l'onglet tradition en mode édition
updateRegionTablesTabForEdit(region) - Met à jour l'onglet tables en mode édition
updateRegionJsonTablesTabForEdit(region) - Met à jour l'onglet tables JSON en mode édition
Gestion des contrôles d'édition :
addRegionEditControls() - Ajoute les boutons de sauvegarde/annulation
setupRegionColorPickerListeners() - Configure les événements pour le sélecteur de couleur
setupRegionTablesEditListeners() - Configure les événements pour l'édition des tables
setupRegionJsonTablesEditListeners() - Configure les événements pour l'édition des tables JSON
addNewRegionTableRow() - Ajoute une nouvelle ligne de table
addNewRegionJsonTableRow() - Ajoute une nouvelle ligne de table JSON
updateRegionTableIndices() - Met à jour les indices des tables
updateRegionJsonTableIndices() - Met à jour les indices des tables JSON
collectRegionTablesFromEdit() - Collecte les données des tables depuis le mode édition
collectRegionJsonTablesFromEdit() - Collecte les données des tables JSON depuis le mode édition
Sauvegarde et annulation :
saveRegionEdit() - Sauvegarde les modifications d'une région
cancelRegionEdit() - Annule les modifications d'une région
showRegionContent(region) - Affiche le contenu d'une région (sans mode édition)
deleteRegion(regionId) - Supprime une région
Génération de contenu :
handleGenerateRegionDescription(event) - Génère une description automatique pour une région
Fonctions pour les lieux :
Affichage et contenu :
showInfoBox(event) - Fonction principale pour afficher les informations d'un lieu
getLocationImages(location) - Récupère la liste des images d'un lieu
getDefaultLocationImage(location) - Récupère l'image par défaut d'un lieu
getLocationTables(location) - Récupère la liste des tables aléatoires d'un lieu
getDefaultLocationTable(location) - Récupère la table par défaut d'un lieu
getLocationJsonTables(location) - Récupère la liste des tables JSON d'un lieu
getDefaultLocationJsonTable(location) - Récupère la table JSON par défaut d'un lieu
Mode édition :
enterEditMode() - Active le mode édition pour un lieu
updateImageTabForEdit(location) - Met à jour l'onglet images en mode édition
updateTextTabForEdit(location) - Met à jour l'onglet texte en mode édition
updateRumeursTabForEdit(location) - Met à jour l'onglet rumeurs en mode édition
updateTraditionTabForEdit(location) - Met à jour l'onglet tradition en mode édition
updateTablesTabForEdit(location) - Met à jour l'onglet tables en mode édition
updateJsonTablesTabForEdit(location) - Met à jour l'onglet tables JSON en mode édition
Gestion des contrôles d'édition :
addEditControls() - Ajoute les boutons de sauvegarde/annulation
setupColorPickerListeners() - Configure les événements pour le sélecteur de couleur
setupTablesEditListeners() - Configure les événements pour l'édition des tables
setupJsonTablesEditListeners() - Configure les événements pour l'édition des tables JSON
addNewTableRow() - Ajoute une nouvelle ligne de table
addNewJsonTableRow() - Ajoute une nouvelle ligne de table JSON
updateTableIndices() - Met à jour les indices des tables
updateJsonTableIndices() - Met à jour les indices des tables JSON
collectTablesFromEdit() - Collecte les données des tables depuis le mode édition
collectJsonTablesFromEdit() - Collecte les données des tables JSON depuis le mode édition
collectImagesFromEdit() - Collecte les données des images depuis le mode édition
setupImageEditListeners() - Configure les événements pour l'édition des images
addNewImageRow() - Ajoute une nouvelle ligne d'image
updateImageIndices() - Met à jour les indices des images
Sauvegarde et annulation :
saveEdit() - Sauvegarde les modifications d'un lieu
cancelEdit() - Annule les modifications d'un lieu
showLocationContent(location) - Affiche le contenu d'un lieu (sans mode édition)
deleteLocation(locationId) - Supprime un lieu
Génération de contenu :
handleGenerateDescription(event) - Génère une description automatique pour un lieu
setupRandomEventButtons(location) - Configure les boutons de génération d'événements aléatoires
